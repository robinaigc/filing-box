# 财报一搜 - 单页网站开发任务书（最终优化版）

**目标**：做一个超级干净、只有一个 HTML 文件的极简财报查询网站。

## 核心要求（必须严格遵守）

- **单文件**：只有一个 `index.html`，包含所有 HTML + Tailwind CSS (CDN) + 纯 Vanilla JavaScript。
- 全部内容都在**一个页面内**，无分页、无路由。
- 极简风格、支持暗黑模式、移动端完美适配。
- 不要使用任何前端框架。

## 页面结构（从上到下）

1. 顶部标题：「财报一搜」 + 暗黑模式切换按钮
2. 大搜索栏（支持 ticker、股票代码、公司名、中英文）
3. 搜索栏下方：**「最近/最热财报推荐」**（默认显示 6 条卡片）
4. 搜索后：在搜索栏下方显示**搜索结果区域**（替换推荐区）

## 搜索逻辑（重点要求）

- 支持**中英文混合模糊搜索**，不区分大小写。
- 输入「苹果」必须能匹配到「苹果公司」或「Apple Inc.」。
- 匹配字段：company（中文名）、ticker、code、title。
- **只显示该公司**的所有财报，**绝不显示其他公司**的结果。
- 找到公司后，表格上方显示公司信息（全称 + ticker/代码 + 市场）。
- 表格内所有报告按**报告日期降序**排序（最新在最上面）。
- 如果没找到，显示醒目提示：“没有找到该公司的财报”。

## 表格列

- 报告日期（YYYY-MM-DD）
- 报告类型（10-K / 10-Q / 年报 / 季报 / 中报 等）
- 报告标题
- 下载（最后一列，点击弹出「另存为」窗口）
